# Gerador IA de personagens para uma campanha de RPG!

Projeto desenvolvido como desafio final da Imersão Inteligência Artificial da **Alura** em parceria com o **Google**. A aplicação utiliza a API do Gemini para fornecer ideias a um futuro jogador de RPG, criando a história de seu personagem se baseando nas escolhas do usuário como raça, alinhamento e classe.
